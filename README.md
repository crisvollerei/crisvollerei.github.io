# 清蒸排班表

一个美观、易用的排班表单页应用，支持自定义排班周期、主题切换、日历视图等功能，适配移动端和桌面端，纯前端实现无需后端依赖。

## ✨ 功能特点

- 🎨 **双主题切换**：支持暗黑模式（默认）和亮色模式，自动记忆用户偏好
- 📅 **今日班次醒目展示**：直观显示今日排班、周期进度、距离下次班次倒计时
- 📆 **日历视图**：按月查看完整排班，支持月份切换，快速回到今日
- 🎯 **自定义配置**：支持设置锚定日期和排班周期模式，配置自动本地保存
- 🎉 **节假日标记**：自动标注重要节假日（元旦、春节、劳动节、国庆节）
- 🔍 **日期详情**：点击任意日期查看该日排班详情和周期位置
- 📱 **响应式设计**：完美适配手机、平板、桌面等各种设备尺寸
- ⚡ **性能优化**：原生JS实现，轻量无框架，加载快速

## 🚀 使用方法

### 1. 克隆仓库
下载index.html即可

### 2. 直接使用
无需安装依赖，直接用浏览器打开 `清蒸排班表.html` 文件即可使用。

### 3. 自定义排班设置
点击页面底部的「⚙️ 排班设置」按钮，可配置：
- **锚定日期**：排班周期的起始参考日期（格式：YYYY-MM-DD）
- **周期模式**：用空格分隔的排班序列，例如：`白班 夜班 休息 白班 夜班 休息 休息 休息`

配置完成后会自动保存到本地存储，刷新页面不会丢失。

## 🛠️ 技术栈

- **HTML5**：语义化标签，结构化布局
- **CSS3**：
  - CSS变量实现主题切换
  - 渐变、阴影、动画提升视觉体验
  - Grid/Flex 布局实现响应式
- **JavaScript (ES6+)**：
  - 原生JS实现所有交互逻辑
  - LocalStorage 本地存储配置
  - 日期处理与计算
- 字体：Noto Sans SC (中文)、JetBrains Mono (等宽字体)

## 🎨 自定义扩展

### 1. 添加更多节假日
修改代码中 `FESTIVALS` 常量，格式：
```javascript
const FESTIVALS = {
  '2026-01-01': '元旦',
  '2026-02-17': '春节',
  // 添加更多节假日...
};
```

### 2. 修改主题配色
修改CSS中 `:root` 和 `[data-theme="light"]` 下的CSS变量：
```css
:root {
  --bg: #0d1117; /* 暗黑模式背景 */
  --accent: #f0883e; /* 强调色 */
  --day-color: #58a6ff; /* 白班颜色 */
  --night-color: #a371f7; /* 夜班颜色 */
  --rest-color: #3fb950; /* 休息颜色 */
  /* 更多变量... */
}
```

### 3. 更换头像
替换代码中 `avatar.png` 为自定义头像图片，或修改头像 fallback 文字：
```html
<img src="your-avatar.png" alt="清蒸" onerror="this.style.display='none'">
<div class="avatar-fallback">清</div>
```
